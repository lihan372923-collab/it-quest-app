内容方向：
- 电脑店技术：电脑组装、系统安装、维修排障、打印机
- 弱电安防：监控、NVR、POE、远程预览、售后排障
- 网络服务器：网络、SSH、Nginx、MySQL、Docker
- 自动化工坊：Python、前端恢复、工作小工具
- 安全训练营：默认密码、端口、日志、安全加固

部署：
```bash
unzip tech-quest-workshop.zip
sudo rm -rf /var/www/it-quest/*
sudo cp -r tech-quest-workshop/* /var/www/it-quest/
```
